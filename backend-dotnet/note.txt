DI in dotnet:
- Using: AddScoped<T> or AddSingleton<T>
    + AddScoped = one scope (http request) one instance
    + AddSingleton = one instance in entire application
- Talking a type <T> and return a value
- If same <T> registed multiple time then only the last services applied:
    builder.Services.AddScoped<string>(sp => "One");
    builder.Services.AddScoped<string>(sp => "Two");
    var s = sp.GetRequiredService<string>();
    Console.WriteLine(s); // Will print "Two"
